## Tablas utilizadas 

1. **actor**: información de los actores 
2. **film**: información de las películas 
3. **film_actor**: relación entre actores y películas 
4. **category**: categorías de películas 
5. **film_category**: relación entre películas y categorías 
6. **inventory**: inventario de películas   
7. **rental**: alquileres de películas
