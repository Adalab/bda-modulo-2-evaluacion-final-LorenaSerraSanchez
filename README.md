## Tablas utilizadas 

1. **actor**: información de los actores (`actor_id`, `first_name`, `last_name`)  
2. **film**: información de las películas (`film_id`, `title`, `length`, `release_year`, `rating`)  
3. **film_actor**: relación entre actores y películas (`actor_id`, `film_id`)  
4. **category**: categorías de películas (`category_id`, `name`)  
5. **film_category**: relación entre películas y categorías (`film_id`, `category_id`)  
6. **inventory**: inventario de películas (`inventory_id`, `film_id`)  
7. **rental**: alquileres de películas (`rental_id`, `inventory_id`, `rental_date`, `return_date`)
